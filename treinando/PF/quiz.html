Quiz 1

  • Para cada problema, o primeiro aluno que enviar o link no JSFiddle com uma solução correta, leva +1pt+1pt+1pt.
  • Um aluno só levará ponto extra para mais de um problema, caso nenhum outro tenha conseguido dentro do tempo, independentemente da fila.
  • Quando o problema apresentar um trecho de solução, este trecho deve ser obrigatoriamente utilizado em sua solução sem qualquer modificação.

⚠️ Os problemas não estão necessariamente em ordem crescente de dificuldade.

Lista de problemas

const data = new Date()
const dia = data.getDate()
const mes = data.getMonth()+1
const ano = data.getFullYear()

  1. Programa para converter temperaturas entre graus Celsius e graus Fahrenheit (e vice-versa). Escreva uma função para cada conversão. Cada função deve realizar a conversão e retornar uma string com o seguinte padrão textual: <valor>°C = <valor>°F (e vice-versa). Depois, mande exibir no console os textos gerados. Exemplo do que será exibido para conversão de 60 graus celsius para fahrenheit: 60°C = 140°F. Dica: o char '\xB0' representa o símbolo °.

  1. Programa para exibir o valor final de um produto após uma dada taxa de desconto ser aplicada (a taxa pode ser representada por um número decimal: 0.1 representa 10%, por exemplo). Você deve definir uma função geral de desconto onde a taxa é um parâmetro e o valor do produto é outro. A seguir, você deve criar funções específicas de desconto (para 10%, 20% e 50% de desconto) reaproveitando-se da aplicação parcial dessa função geral.
